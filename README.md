# Piddle

A bold streak-tracking app with a modern wellness twist — built to help users quit wasting time and start stacking habits.

## Design Philosophy

- **Brand Colors**: Deep red background, striking red hero tone, with clean white and black tertiary colors.
- **Typography**: Fredoka One for headings and Nunito for body text.
- **Visual Identity**: Strong logo mark, polished rounded UI, elevated shadows, and clear brand personality.
- **Tone**: Playful, energetic, and confident — not childish, but definitely not boring.

## Features

- **Custom Streaks**: Create personalized streaks with names, descriptions, categories, icons, colors, and privacy settings.
- **Gamified Tracking**: Visual streak counters with animated fire emojis, current and longest streaks.
- **Daily Check-ins**: Satisfying animations and confetti celebrations for milestones (7, 30, 100 days).
- **Urgency Cues**: Warnings for unchecked streaks with time until midnight.
- **Friends System**: Add friends, view their public streaks, and send cheer reactions.
- **Public/Private Toggle**: Control visibility of your streaks.
- **Hall of Fame**: Milestone celebrations for hitting big numbers.
- **Notifications Placeholder**: Bell icon for future reminders.
- **Responsive Design**: Bright, energetic UI with animations and gradients.

## Prerequisites

- Node.js 18+ (install from [https://nodejs.org/](https://nodejs.org/))

## Getting Started

1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Tech Stack

- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- Framer Motion for animations
- React Confetti for celebrations
- Local Storage for data
